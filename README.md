Real-Time Weather Forecasting System

Project Overview

This project is a web-based weather forecasting application that provides real-time weather information using API integration. Users can search for a city and view current weather details such as temperature, humidity, wind speed, and weather conditions through a simple and responsive interface.

Features

- Real-time weather updates
- Search weather by city name
- Displays temperature, humidity, and wind speed
- Dynamic weather condition display
- Responsive user interface
- API integration for live weather data
- Error handling for invalid inputs

Technologies Used

- HTML
- CSS
- JavaScript
- REST API
- OpenWeatherMap API

Working

1. User enters a city name.
2. The application sends a request to the weather API.
3. Weather data is retrieved in JSON format.
4. The application displays temperature, humidity, wind speed, and weather conditions.
5. The interface updates dynamically based on the fetched data.

How to Run

1. Clone the repository:

git clone https://github.com/vasavi-pachigolla/real-time-weather-forecasting-system.git

2. Open the project folder.

3. Add your API key in the JavaScript file.

4. Open index.html in a web browser.
